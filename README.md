# AI Mentor Agent

An AI-powered mentor built using Google's Agent Development Kit (ADK).

## Features

* AI engineering guidance
* Programming assistance
* Learning roadmaps
* Project recommendations
* Career mentoring

## Tech Stack

* Google ADK
* Gemini 2.5 Flash
* Python
* YAML Agent Configuration

## Run Locally

```bash
pip install google-adk
adk web
```
